Markdown.Blog

Project Goals

The goal of this project is to establish a universal Markdown-based blogging solution, which can also be extended to general content management. It focuses on summarizing rules, methods, and best practices for reusability.

Features

In the initial phase, the project aims to develop reusable components for editing and displaying articles using Markdown code, reducing the need for repetitive code development. Key areas of focus include:

  • Content Editing
  • Tag Management
  • Category Management
  • Image Management and Utilization
  • Integration with External Links and Specific Service Providers (e.g., YouTube for videos, social media embeds)
